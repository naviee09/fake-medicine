# Natural Medicine Authenticity Verifier

An AI-powered platform designed to help users verify the authenticity of natural medicines using image analysis and optional identifier-based checks. The system leverages deep learning and computer vision techniques to analyze medicine packaging and detect visual inconsistencies that may indicate counterfeit products.

## Features

- User authentication with secure login
- AI-powered image analysis for packaging verification
- Optional identifier-based authentication
- Natural product focus for health protection
- Project review page with comprehensive information
- User-friendly interface with responsive design
- Real-time verification results

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Building for Production

```bash
npm run build
```

### Linting

```bash
npm run lint
```

## Usage

1. Sign in to access the verification system
2. Upload an image of the natural medicine packaging
3. Optionally, enter an identifier (batch number, barcode, etc.)
4. Click "Verify Authenticity" to analyze the image
5. View the verification results
6. Visit the Project Review page for detailed information about the platform
